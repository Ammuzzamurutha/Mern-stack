==>Alpha and opacity
alpha deals with color visibility
rgba(255,255,0,0.5) ranges from 0-1
opacity deals with both color and element visibility
opacity=0.6 ranges from 0-1
==>css transitions property
changing from one state to another state
transition: margin-top 2s ease-in-out 0.2s; // property name |duration|timing-function|delay
==>css transform 
helps to rotate scale skew or translate an element
transform:rotate(45deg)
transform:scale(1.5) 
transform:translate(20px,20px)  
transform:skew(30deg)
